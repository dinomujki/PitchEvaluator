Install Instruction:
--------------------

1. Clone the repository to your own computer.
2. Open the terminal and cd into the root of the repository.
3. Run:
  ```
  npm install
  ```
   then run:
  ```
  npm start
  ```
4. Open a browser and go to this url:
```
http://localhost:8000/
```
