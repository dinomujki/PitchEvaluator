Install Instruction:
--------------------

1. Clone this repository to your own computer.
2. Open terminal and cd to the project file.
3. Run:
  ```
  npm install
  ```
   then run:
  ```
  npm start
  ```
4. Open browser go to url:
```
http://localhost:8000/app/#/view1
```
